# Slang Practice

`Slang Practice` 是 RTR2 文档站中的练习板块，用来沉淀基于本仓库样例的 Slang 学习路径。

这个板块的目标不是覆盖全部 Slang 特性，而是围绕当前工程能直接运行、直接修改、直接观察结果的案例，逐步建立稳定的 shader 编写能力。

## 板块结构

当前规划如下：

- `Slang Basics`
  - 第 1 节：`editable_shadertoy`、Slang 最小语法与 compute shader 计算模型

后续可以在这个板块下继续增加：

- 更多 `Slang Basics` 小节
- 资源绑定与纹理采样
- 多 pass 练习
- 更复杂的 ShaderToy 风格效果拆解

## 学习建议

推荐按下面的顺序阅读和练习：

1. 先看 `Slang Basics` chapter 的概览页，明确当前 chapter 要解决什么问题。
2. 再进入第 1 节讲义，跟着 `editable_shadertoy` 跑通最小闭环。
3. 每完成一节，都把示例跑起来并亲手改一次 shader，不要只读文档。
